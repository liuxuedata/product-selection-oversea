# product-selection-oversea

基于跨境站选品

## 项目结构

- `frontend/` - 基于 Next.js 的前端应用。
- `backend/` - 基于 Express 的后端服务。

## 开发

### 前端

```bash
cd frontend
npm install
npm run dev
```

### 后端

```bash
cd backend
npm install
npm start
```

### 代码格式化

在项目根目录运行：

```bash
npm test
```

该命令使用 Prettier 检查代码格式。
